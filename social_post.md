## LinkedIn Post

Excited to share our Elasticsearch Agent Builder Hackathon submission: the DCO Threat Triage Agent.

The problem: SOC analysts drown in alerts. 68% report alert fatigue, and real threats slip through the cracks while teams chase false positives.

Our solution: An autonomous AI agent built with Elastic Agent Builder that performs first-pass triage in 30 seconds instead of 30 minutes. It uses four ES|QL threat-hunting tools and semantic search against a MITRE ATT&CK-mapped IOC database to correlate events, detect C2 beaconing, trace lateral movement, and generate structured triage reports with containment recommendations.

We simulated a realistic 5-stage attack chain buried in 80+ noise events to demonstrate real-world purple team triage capability.

Built with: Elastic Agent Builder, ES|QL, Elasticsearch hybrid search, MITRE ATT&CK, Claude Agent SDK, Python.
