## LinkedIn Post

Excited to share our Elasticsearch Agent Builder Hackathon submission: the **DCO Threat Triage Agent**.

**The problem:** SOC analysts drown in alerts. 68% report alert fatigue, and real threats slip through the cracks while teams chase false positives.

**Our solution:** An autonomous AI agent built with Elastic Agent Builder that performs first-pass triage in 30 seconds instead of 30 minutes. It orchestrates 7 purpose-built tools -- 5 ES|QL threat-hunting queries, a hybrid search tool for MITRE ATT&CK-mapped IOCs, and a workflow tool for automated incident logging -- to correlate events, detect C2 beaconing patterns (~345s callback intervals), trace lateral movement across hosts, and generate structured triage reports with containment recommendations.

We simulated a realistic 5-stage MITRE ATT&CK attack chain (T1566 phishing through T1041 exfiltration) buried in 80+ noise events to demonstrate real-world purple team triage capability.

Built with: Elastic Agent Builder, ES|QL, Elasticsearch hybrid search, MITRE ATT&CK, Next.js 14, Python.
